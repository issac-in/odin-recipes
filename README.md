# odin-recipes
This is a basic recipe website that links to three meal prep recipes. It was created using HTML so that I could demonstrate the HTML knowledge I've acquired so far from 'The Odin Project', and my familiarity with Git.

Challenges I faced was how to go about adding the Fahrenheit symbol to the content, and how to best structure my HTML files semantically for the sake of accessibility.

I hope to add design and interactivity to this website in the future with CSS and Javascript in the future.
